User Software & Computing

The goal of USCMS Software and Computing (USCMSSC) project is to provide the software and computing resources needed to enable US physicists to fully participate in the physics program of CMS. To realize this goal, US physicists should play a key role in all stages of computing related activities, from development of the reconstruction software and computing infrastructure to the examination of physics results.

The central CMS software repository and distribution is at CERN. For more information, visit the CMS CCS Computing and Core Software Home at CERN.

The objective of USCMSSC web pages is to provide most current information to the USCMSSC team, including collaborators, about:

User Computing, including information about establishing accounts, computing setups, tutorials and the components of CMS software

CMS Software, distribution, associated software tools and related documentation

U.S. CMS Grid, information about the deployment, development and on-going management of the grid environment for USCMS

Project Management information including work breakdown structure (WBS), funding resources, reviews, and various project related documents
