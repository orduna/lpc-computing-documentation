Getting Started: Get an Account at Fermilab

How to get a CMS Computing Account at Fermilab

Remote Users:

If you are a USCMS Computing Remote user and fall into one of the following categories:

  • Never had FERMI or CMS computing privileges
  • Had Fermi Computing privileges but not CMS computing privileges
  • Need to renew your Fermilab Kerberos credentials,

use the following steps:

  • Read the information about getting started as a non-employee off-site Fermilab user. This is where you will apply for your accounts.
  • On the application form, select "E-892/919 (CMS)" as your experiment
    and complete the form. You will get your CMS computing privileges. You will receive e-mail notification, and if requested, a cryptocard will be mailed to you.

If you are a remote user but already have a FERMI ID do the following to request an account on the CMS Computing systems:

  • Log into the Fermilab Service Desk (Your Services account password is required)
  • Select the "Service Request Catalog" link in the left frame
  • Select Affiliation\Experiment Computing Account Request from the "Accounts" box. (The form should be auto-filled from your info.)
  • Select CMS from the pulldown menu under "Select Affiliation/Experiment" as shown in the following image:
  • Complete the remaining fields and submit the form.

To renew your Fermilab Kerberos credentials follow the directions on this page under the heading Renew Accounts.

On-Site Users:

If you are an on-site user (employee, visitor or contractor working locally) and already have a valid Fermilab ID and a services account please do the following:

  • Log into the Fermilab Service Desk
  • Select the "Service Catalog" link in the left frame
  • Select Affiliation\Experiment Computing Account Request from the "Accounts" box. (The form should be auto-filled from your info.)
  • Submit the form.

If you are a new on-site user you need to obtain a Fermilab ID and computer accounts following the directions on this page for new visitors or employees.
